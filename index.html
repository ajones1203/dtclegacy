🤝

Collaboration streams let multiple creators go live together. Each person appears split-screen with their own video feed. Ideal for interviews, panel discussions, business partnerships, storytelling sessions, and TC Shop product demos with other sellers.

How it works
1️⃣
Host goes live from the Host tab
Start your stream and click Invite Co-Host. A unique stream code is generated.
2️⃣
Share your stream code
Send the code to your collaborators via DM or any channel. Up to 3 co-hosts can join.
3️⃣
Co-hosts join from the Join tab
Each co-host enters the code and their camera appears in a split-screen panel alongside the host.
4️⃣

When the host ends the stream, the full multi-host session is archived to the host's capsule.
